# 计代源作品集

个人摄影作品集网站

## 技术栈

- HTML5
- CSS3
- JavaScript

## 功能特性

- 响应式设计
- 作品分类筛选
- 图片弹窗浏览
- 明暗主题切换

## 部署方式

1. 克隆仓库：
```bash
git clone git@github.com:JJin-Photo/My-Personal-Portfolio.git
```

2. 进入项目目录：
```bash
cd My-Personal-Portfolio
```

3. 使用GitHub Pages部署：
   - 在GitHub仓库设置中启用GitHub Pages
   - 选择主分支作为源

## 项目结构

```
├── index.html          # 首页
├── works.html          # 作品集页面
├── styles.css          # 样式文件
├── script.js           # 交互脚本
├── 剧照/               # 剧照作品
├── 摄影/               # 摄影作品
└── 证件照/             # 证件照
```
